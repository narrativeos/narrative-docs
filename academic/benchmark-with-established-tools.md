# Benchmark with Established Tools

本页用于说明 NarrativeOS 与 VOSviewer/CiteSpace 的互补关系。

## 对照原则

- 不做“全面替代”表述
- 仅比较研究任务层面的适配度
- 明确联合使用场景

## 互补矩阵（摘要）

| 场景 | VOSviewer/CiteSpace | NarrativeOS |
|---|---|---|
| 文献关系网络构建 | 强项 | 非核心目标 |
| 叙事结构诊断与解释 | 辅助 | 强项 |
| 研究流程证据链复核 | 部分支持 | 强调完整链路 |
| 论文方法与输出打包 | 需要额外整理 | 提供模板化支持 |

## 联合使用建议

1. 先用文献计量工具定位主题簇和前沿。
2. 再用 NarrativeOS 对关键文本簇做叙事结构深描。
3. 用统一复现包整合两类证据，支撑论文论证。

## 非目标说明

- 本页不评价工具优劣排名
- 本页不构成性能基准报告
