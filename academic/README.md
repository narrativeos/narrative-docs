# Academic Research Track

## 摘要（中文）

本分区面向研究者提供一条可验证的采用路径：先建立方法信任，再完成最小可复现任务，最后进入论文与开源协作流程。

## Executive Summary (EN)

This section provides a research-oriented adoption path for NarrativeOS, focusing on trust, reproducibility, publication readiness, and open collaboration.

## 本分区边界

- 本分区关注研究方法、可复核流程和学术采用。
- 本分区不替代算法实现文档，底层实现仍以 architecture 与 developer 分区为准。
- 本分区与 VOSviewer/CiteSpace 的关系定义为“互补使用场景说明”，不是替代声明。
- 本分区采用 Kernel First：先定义底层不可变方法规则，再通过 profile 进行场景化配置。

## Kernel First 阅读提示

再按研究目标进入具体示例。
