# User 用户

本页面向实际使用 NarrativeOS 的读者，而不是文档维护者。

如果你只是想完成一次从文本导入到诊断结果核对的最小闭环，应先读 Getting Started，再按场景进入 Editor、Atlas、Cloud 或 GIS。

## 推荐阅读顺序

- 第一步：先看 [user/README.md](user/README.md) 了解用户文档分区
- 第二步：再看 [user/getting-started/README.md](user/getting-started/README.md) 完成首个成功任务
- 第三步：按场景进入 Editor、Atlas、Cloud 或 GIS 相关章节
