## 定位 | Positioning

Copilot 不应仅作为补全工具，而应作为 Dev Agent 参与：

- 实现
- 重构
- 测试补全
- 文档维护

## 边界

- 架构与产品边界由人定义
- AI 输出必须通过规则与 CI 验证
