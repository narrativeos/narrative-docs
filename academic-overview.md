# Academic 学术研究

本页面面向高校研究者、课题组成员和方法论评审者。

它用于回答三个问题：NarrativeOS 在研究流程中解决什么问题、结果如何复核、如何把结果带入论文与开源协作。

## 推荐阅读顺序

- 第一步：先看 [academic/README.md](academic/README.md) 了解学术分区边界
- 第二步：再看 [academic/trust-methodology.md](academic/trust-methodology.md) 与 [academic/reproducibility-kit.md](academic/reproducibility-kit.md) 建立方法信任
- 第三步：进入 [academic/10-minute-first-success.md](academic/10-minute-first-success.md) 完成首个可验证任务
- 第四步：按研究目标选择 [academic/research-workflows.md](academic/research-workflows.md) 与 [academic/publication-support.md](academic/publication-support.md)
