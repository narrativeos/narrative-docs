# Code of Conduct

## EN Summary

This document defines expected behavior, unacceptable behavior, and enforcement process for contributors and maintainers in the NarrativeOS documentation community.

## 社区承诺

我们致力于建设一个专业、尊重、包容、可协作的开源文档社区。

所有参与者都应以“改进事实质量与协作效率”为共同目标。

## 期望行为

- 尊重不同背景、经验与观点
- 基于事实、证据和上下文讨论问题
- 对反馈保持建设性与可执行性
- 在评审中聚焦问题本身，而非个人标签
- 主动修复自己引入的错误与不一致

## 不可接受行为

- 侮辱、骚扰、歧视、威胁或人身攻击
- 持续发布与讨论主题无关的破坏性内容
- 泄露他人隐私、敏感信息或未公开安全细节
- 恶意曲解、反复挑衅或报复性行为
- 明知错误仍拒绝修复并阻碍协作

## 适用范围

本准则适用于：

- 仓库 Issue、PR、Discussion 与评审评论
- 与本仓库协作直接相关的同步沟通场景

## 执行机制

- Maintainers 负责接收与处理违规报告
- 可采取措施包括：提醒、警告、临时限制、移除协作权限
- 对严重或反复违规行为，可直接采取更严格处置

## 举报与处理

若你遭遇或观察到违规行为：

- 请向维护者进行私密举报，并提供必要证据
- 维护者将尽快确认并进行审查
- 处理过程尽量保护举报人隐私并避免二次伤害

## 复议机制

被处理方可在合理期限内提出复议，维护者将进行复核并给出最终结论。
