## Atlas 页面定位

Atlas 是语言地图核心页面，默认以全屏方式进入。

使用目标是观察文本内部结构，而非仅进行顺序阅读。

## 五种地图模式

### 1. City Mode

- 文章视为城市结构
- 段落视为街区
- 段间关联视为道路
- 用于识别结构断桥与组织断点

### 2. Galaxy Mode

- 词语与概念视为星体网络
- 点击关键词可展开关联主题
- 用于追踪概念聚类与关系路径

### 3. Music Mode

- 句长、节奏、停顿以波形方式呈现
- 用于定位阅读负担段与节奏突变段

### 4. Emotion Mode

- 情绪变化以连续色带或河流形式展示
- 用于观察叙事情绪走向

### 5. X-Ray Mode

- 直接显示问题扫描结果
- 重点暴露冗余、断桥、AI 模板、过密区
- 用于编辑优先修订

## 推荐操作流程

1. 先用 City Mode 建立结构感知
2. 切换 Galaxy Mode 观察主题关系
3. 用 Music/Emotion Mode 检查节奏与情绪连续性
4. 最后用 X-Ray Mode 收敛问题并回跳原文修订
